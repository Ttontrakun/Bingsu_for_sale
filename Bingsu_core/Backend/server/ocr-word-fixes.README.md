# รายการคำผิด OCR (ocr-word-fixes.json)

ใส่คำที่ OCR อ่านผิดแล้วต้องการให้แทนที่อัตโนมัติ โดยไม่ต้องแก้โค้ด

**รูปแบบ:** array ของคู่ `[ "คำผิด", "คำถูก" ]`

ตัวอย่าง:

```json
[
  ["ท วาม", "ความ"],
  ["มหาวิท ยาลัย", "มหาวิทยาลัย"],
  ["บรรณำนุกรม", "บรรณานุกรม"]
]
```

- แทนที่แบบ **literal** (ไม่ใช้ regex) — ใส่ตัวอักษร/ช่องว่างตามที่ OCR ออกมาได้เลย
- ไฟล์อยู่ที่ `server/ocr-word-fixes.json` — แก้แล้ว request ถัดไปจะใช้รายการล่าสุด (ไม่ต้อง restart)
